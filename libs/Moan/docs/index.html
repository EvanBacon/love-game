Module Moan

speak (title, messages, config)
Message instance constructor

Parameters:

  • title string Title box text, default colour
  • title:(table)
    • [1]: (string) Messagebox title text
    • [2]: (table) Messagebox title colour; {255, 100, 10}
  • messages table
  • config table

Usage:

    avatar = love.graphics.newImage("image.png")
Moan.speak({"Mike", {0,255,0}}, {"Message one", "two--and", "Here's those options!"}, {x=10, y=10, image=avatar,
                  onstart=function() camera:move(100, 20) end, oncomplete=function() Moan.setSpeed("slow") end,
                  options={
                   {"Option one",  function() option1() end},
                   {"Option two",  function() option2() end},
                   {"Option three",function() option3() end}}
                   {"Option n...", function() optionn() end}}
                  })

Tables

Moan
Main configuration table

Fields:

  • indicatorCharacter string Next message indicator, (default ">")
  • optionCharacter string Option selection prefix, (default "- ")
  • indicatorDelay int Time between next message indicator being toggled, (default 25)
  • selectButton string Key that advanced to next message, (default "space")
  • typeSpeed int Delay between character being printed via the typewriter
  • debug bool Display debug information
  • allMsgs table Contains messages (if not overriden by Moan.defMsgContainer())
  • history table Contains all messages that have been called
  • currentMessage string Current full message
  • currentMsgInstance int Current Moan.speak() instance+
  • currentMsgKey int Current key of current instances messages table
  • currentOption int Current key of selected option table
  • userdata currentImage Current messagebox Love image object
  • _VERSION string Moan version
  • _URL string Moan repo url
  • _DESCRIPTION string Short description of the library
  • UI table UI configuration
